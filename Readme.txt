VRInput by faidra

◆なにこれ
ざっくりいうと、VIVEコントローラーで日本語入力をするためのものです。

◆ざっくりとした使い方
左手のパッドタッチで「ひらがな」「濁点」「特殊入力」などを決め、
右手のパッドタッチで「あ行」「か行」などを決め、
右手のひねり&振りで「あいうえお」などを入力します。

◆リファレンス
- 通常文字
右手のパッドの中央をタッチしたまま、右手を振ることで、「あ」「い」「う」「え」「お」が出せます。
振り方はあいうえおの順番に「反時計回りにひねる」「左へ振る」「上へ振る」「右へ振る」「下へ振る」です。 (早めにシュッとやる感じ)
同様に、右手のパッドの左側をタッチしたまま振ると「か行」が出ます。
右パッド上で「さ行」
右パッド右で「た行」
右パッド下で「な行」
左パッド中央&右パッド中央で「は行」
左パッド中央&右パッド左で「ま行」
左パッド中央&右パッド上で「や行」「っ」「ー」
左パッド中央&右パッド右で「ら行」
左パッド中央&右パッド下で「わ行」「。」「、」
左パッド右&右パッド中央で「ば行」
左パッド右&右パッド左で「が行」
左パッド右&右パッド上で「ざ行」
左パッド右&右パッド右で「だ行」
左パッド右&右パッド下で「ゃ行」(小さいや行)
左パッド上&右パッド中央で「ぱ行」
が出ます。

- 特殊入力
左パッド左をタッチしたまま、右パッドの中央をタップするとスペースが出ます (日本語入力中なら変換になる)
同様に、左パッド左をタッチしたまま、右パッド左をタップでバックスペース
左パッド左&右パッド上タップで半角/全角
左パッド左&右パッド右タップでエンター
左パッド左&右パッド下タップでスペース
になります。

◆Q&A
- デスクトップ表示モードで使えないんだけど
仕様です。あの状態でコントローラーの入力を取る方法が分かりませんでした。
VRアプリ内の入力に使う・オーバーレイツール(OVRDropなど)への入力に使う・VIVEコンを持ちながらリアルモニタを見て使う、などの用途になります。

- VIVE以外では使えないの？
試してませんが、今は使えない気がします

- Windows以外では使えないの？
試してませんが、サポートする可能性は非常に低いです (IMEの状態取りたいため)

- このキーは出ないの？
現状は出ません。

- 使いにくいんだけど
今は諦めてください。ちょっとずつなんとかします

- デスクトップに出る表示とズレる
仕様です。諦めてください。
